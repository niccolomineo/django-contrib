# django-contrib

A dockerized project for Django contributions.

## Usage

First, clone your `django` fork:

```
$ git clone git@github.com:<USERNAME>/django.git
```

That's all you have to do. You can now run the container(s).

Happy django contrib!


## Testing

Run the test suite:

```
$ cd django
$ make test
```

A custom settings module in `settings/test_local.py` is passed by default.
